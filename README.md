# Superstore Sales Dashboard

## Project Overview
This project analyzes retail sales data using Tableau to identify revenue trends, top-performing products, and category performance.

## Tools Used
- Tableau
- Excel/CSV dataset

## Key Insights
- Technology category generates the highest revenue.
- A small set of products contributes major sales.
- Revenue shows monthly variation indicating seasonality.

- Tableau Dashboard (.twbx)
